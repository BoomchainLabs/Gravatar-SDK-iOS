#  AvatarURL Calculator
Create and validate Gravatar image URLs


## Overview

To create a valid Gravatar URL to fetch avatars, use the ``AvatarURL`` type.
You can create and validate an AvatarURL from a **known Gravatar image URL**, from an **email**, or an **email hash string**.

### Creating an AvatarURL from a known Gravatar URL

```swift
/// Create an AvatarURL from a known Gravatar URL
let url = URL(string: "https://gravatar.com/avatar/6762(...)e674")!
let options = AvatarQueryOptions(
    // The calculated final size will depend on the scale factor
    preferredSize: .points(80)
)

if let avatarURL = AvatarURL(url: url, options: .init(preferredSize: .points(80))) {
    print("Canonical URL: \(avatarURL.canonicalURL)")
    // Canonical URL: https://gravatar.com/avatar/6762(...)e674
    print("Avatar hash: \(avatarURL.hash)")
    // 6762(...)e674
    print("Full URL with query options: \(avatarURL.url)")
    // https://gravatar.com/avatar/6762(...)e674?s=240
}
```

### Creating an AvatarURL from an Identifier

An identifier can be a user email or its hash string. (Check ``AvatarIdentifier``)

```swift
/// Create an AvatarURL from an AvatarIdentifier and apply custom options
let identifier = AvatarIdentifier.email("some@email.com")
let options = AvatarQueryOptions(
    preferredSize: .points(128),
    rating: .general,
    defaultAvatarOption: .monsterId
)

if let avatarURL = AvatarURL(with: identifier, options: options) {
    print("Created URL: \(avatarURL.url)")
    // https://gravatar.com/avatar/6762(...)e674?d=monsterid&s=384&r=g  
}
```

### Update existing options from an AvatarURL

```swift
let identifier = AvatarIdentifier.email("some@email.com")
let options = AvatarQueryOptions(
    preferredSize: .points(128),
    rating: .general,
    defaultAvatarOption: .monsterId
)

if let avatarURL = AvatarURL(with: identifier, options: options) {
    print("URL: \(avatarURL.url)")
    // URL: https://gravatar.com/avatar/6762(...)e674?d=monsterid&s=384&r=g

    let newOptions = AvatarQueryOptions(preferredSize: .points(80))
    let updatedAvatarURL = avatarURL.replacing(options: newOptions)

    print("Updated URL: \(updatedAvatarURL?.url.absoluteString ?? "nil")")
    // Updated URL:  https://gravatar.com/avatar/6762(...)e674?s=240
}
```
